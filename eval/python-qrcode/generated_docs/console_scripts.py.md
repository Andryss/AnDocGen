# Модуль `console_scripts.py`


Модуль для создания QR-кодов из стандартного ввода или первого аргумента. Если вывод в терминал, QR-код будет отображен там; если вывод в файл через канал, будет создан изображение в формате PNG.

**Содержание:**

- [Функции](#функции)

## Функции

### `def commas(items: Iterable[str], joiner = 'or') -> str`

Функция для создания строки с разделенными запятыми элементами.

**Параметры:**

- `items` (`Iterable[str]`) — список строковых элементов
- `joiner` (`str, optional`) — строка, которая будет использоваться как соединитель между последним двумя элементами (по умолчанию 'or')

**Возвращаемое значение:**

- `str` — строка с разделенными запятыми элементами и использованием указанного соединителя

**Примеры:**

```python
print(commas(["apple", "banana", "cherry"]))  # apple, banana or cherry
print(commas(["apple"], joiner="and"))         # apple
print(commas([]))                                # 
```

### `def get_factory(module: str) -> type[BaseImage]`

Возвращает тип изображения-фабрики по полному пути модуля.

**Параметры:**

- `module` (`str`) — полный путь к модулю, содержащему фабрику изображений

**Возвращаемое значение:**

- `type[BaseImage]` — тип фабрики изображений

**Исключения:**

- `ValueError` — если указан неполный путь к модулю

**Примеры:**

```python
factory = get_factory("console_scripts.ImageFactory")
```

### `def get_drawer_help() -> str`

Функция для получения помощи по использованию различных типов ящиков.

**Возвращаемое значение:**

- `str` — строка с информацией о доступных типах ящиков и способах их использования

### `def main(args = None)`

Функция `main` используется для создания QR-кода из переданных аргументов и вывода его в указанный файл или поток.

**Параметры:**

- `args` (`list[str], optional`) — список аргументов командной строки; если не указан, берутся аргументы из `sys.argv[1:]`

**Исключения:**

- `ValueError` — если указан неполный путь к модулю при использовании опции `--factory`
- `TypeError` — если указанный модуль не содержит фабрику изображений
- `optparse.OptionError` — если указан некорректный аргумент командной строки

**Побочные эффекты:**

Функция изменяет поток стандартного вывода и файлы на диске, если указывается параметр `--output`.

**Примеры:**

```python
# Пример вызова функции с аргументами из стандартного ввода
main()

# Пример вызова функции с конкретными аргументами
main(["example_data"])
```
