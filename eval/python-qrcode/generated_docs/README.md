# python-qrcode

Документация проекта, сгенерированная AnDocGen.

## Корневые модули

| Модуль | Описание |
|--------|--------------|
| [__init__.py](__init__.py.md) | Инициализационный модуль для библиотеки создания QR-кодов. |
| [LUT.py](LUT.py.md) | Нет доступного кода для документации. |
| [__main__.py](__main__.py.md) | N/A |
| [base.py](base.py.md) | Модуль содержит базовые константы и классы для работы с генерацией QR-кодов. |
| [console_scripts.py](console_scripts.py.md) | Модуль для создания QR-кодов из стандартного ввода или первого аргумента. Если вывод в терминал, QR-код будет отображен там; если вывод в файл через канал, будет создан изображение в формате PNG. |
| [constants.py](constants.py.md) | Модуль, содержащий различные константы. |
| [exceptions.py](exceptions.py.md) | Модуль, содержащий пользовательские исключения. |
| [main.py](main.py.md) | Модуль для создания и работы с QR-кодами. |
| [release.py](release.py.md) | Этот модуль предоставляет функции для работы с zest.releaser при выпуске новых версий библиотеки qrcode. |
| [util.py](util.py.md) | Модуль с вспомогательными функциями и классами для работы с QR-кодами. |
