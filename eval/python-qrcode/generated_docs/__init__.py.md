# Модуль `__init__.py`


Инициализационный модуль для библиотеки создания QR-кодов.

**Экспорт:**

- `ERROR_CORRECT_H` (`int`) — Высокий уровень коррекции ошибок.
- `ERROR_CORRECT_L` (`int`) — Низкий уровень коррекции ошибок.
- `ERROR_CORRECT_M` (`int`) — Средний уровень коррекции ошибок.
- `ERROR_CORRECT_Q` (`int`) — Очень высокий уровень коррекции ошибок.
- `QRCode` (`class`) — Класс для создания QR-кодов.
- `image` (`module`) — Модуль для работы с изображениями.
- `make` (`function`) — Функция для создания QR-кода.
- `run_example` (`function`) — Функция для запуска примера.

**Содержание:**

- [Функции](#функции)

## Функции

### `def run_example(data = 'http://www.lincolnloop.com', *args, **kwargs)`

Builds an example QR Code and displays it.

**Параметры:**

- `data` (`str`) — данные для кодирования в QR Code (по умолчанию 'http://www.lincolnloop.com')
- `*args` — дополнительные позиционные аргументы, передаваемые конструктору QRCode
- `**kwargs` — дополнительные именованные аргументы, передаваемые конструктору QRCode

**Побочные эффекты:**

Отображает QR Code в графическом интерфейсе пользователя.

**Примеры:**

```python
run_example('https://example.com')
```

**Смотрите также:**

- `make` — упрощенный способ создания QR Code
