### `class Polynomial`

Класс для представления полиномов в контексте криптографии.

#### Методы

##### `__getitem__(index)`

Возвращает элемент многочлена по заданному индексу.

**Параметры:**

- `index` (`int`) — индекс элемента многочлена

**Возвращаемое значение:**

- `float` — значение элемента многочлена

**Примеры:**

```python
p = Polynomial([1, 2, 3])
print(p[0])  # Выведет: 1.0
print(p[1])  # Выведет: 2.0
```

##### `__init__(num, shift)`

Инициализирует объект класса `Polynomial` с коэффициентами и смещением.

**Параметры:**

- `num` (`list`) — список коэффициентов полинома
- `shift` (`int`) — смещение

**Исключения:**

- `ValueError` — если длина списка `num` равна 0

**Граничные случаи:**

- Если `num` пустой, вызывается исключение `ValueError`.
- Если все коэффициенты в начале списка равны нулю и `shift` не равно нулю, коэффициенты корректируются с учетом смещения.

**Примеры:**

```python
p = Polynomial([1, 2, 3], 1)
print(p.num)  # [2, 3, 0]
```

##### `__iter__()`

Возвращает итератор для объекта Polynomial.

**Возвращаемое значение:**

- `iter` — итератор для объекта Polynomial

**Примеры:**

```python
# Пример использования
poly = Polynomial([1, 2, 3])
for coeff in poly:
    print(coeff)
```

##### `__len__()`

Возвращает количество коэффициентов в многочлене.

**Возвращаемое значение:**

- `int` — количество коэффициентов в многочлене

**Примеры:**

```python
# Пример использования метода __len__
poly = Polynomial([1, 2, 3])
print(len(poly))  # Вывод: 3
```

##### `__mul__(other)`

Умножает текущий полином на другой и возвращает новый полином.

**Параметры:**

- `other` (`Polynomial`) — другой полином, на который умножается текущий

**Возвращаемое значение:**

- `Polynomial` — произведение текущего полинома и `other`

**Примеры:**

```python
poly1 = Polynomial([1, 2, 3], 0)
poly2 = Polynomial([4, 5, 6], 0)
result = poly1 * poly2  # result будет эквивалентен Polynomial([4, 13, 28, 27], 0)
```

##### `__mod__(other)`

Выполняет операцию взятия остатка от деления многочлена на другой многочлен и возвращает результат.

**Параметры:**

- `other` (`Polynomial`) — другая многочлен, на которую выполняется операция взятия остатка

**Возвращаемое значение:**

- `Polynomial` — многочлен, являющийся остатком от деления текущего многочлена на `other`

**Примеры:**

```python
poly1 = Polynomial([1, 2, 3], 0)  # x^2 + 2x + 3
poly2 = Polynomial([4, 5], 0)     # 4x + 5
result = poly1 % poly2            # Остаток от деления poly1 на poly2
print(result)                      # Output: Polynomial([3, -7, 38], 0)
```

## Функции

### `def gexp(n)`

Функция `gexp` возвращает значение из таблицы `EXP_TABLE` по индексу, вычисленному как остаток от деления входного параметра `n` на 255.

**Параметры:**

- `n` (`int`) — входной параметр для вычисления индекса в таблице `EXP_TABLE`

**Возвращаемое значение:**

- `int` — значение из таблицы `EXP_TABLE` по вычисленному индексу

**Примеры:**

```python
result = gexp(256)  # result будет равно значению EXP_TABLE[1]
```

### `def glog(n)`

Функция `glog` возвращает значение из таблицы логарифмов для входного числа.

**Параметры:**

- `n` (`int`) — число, для которого нужно получить значение из таблицы логарифмов

**Возвращаемое значение:**

- `float` — значение из таблицы логарифмов для числа `n`

**Исключения:**

- `ValueError` — если `n` меньше 1

**Примеры:**

```python
result = glog(2)  # Возвращает значение из таблицы логарифмов для числа 2
```

### `def rs_blocks(version, error_correction)`

Создает блоки Reed-Solomon и возвращает их список.

**Параметры:**

- `version` (`int`) — версия QR-кода
- `error_correction` (`constants.ERROR_CORRECT_L | constants.ERROR_CORRECT_M | constants.ERROR_CORRECT_Q | constants.ERROR_CORRECT_H`) — уровень исправления ошибок

**Возвращаемое значение:**

- `list[RSBlock]` — список блоков Reed-Solomon

**Исключения:**

- `ValueError` — если не поддерживается указанная версия и уровень исправления ошибок

**Примеры:**

```python
rs_blocks(4, constants.ERROR_CORRECT_L)
```
