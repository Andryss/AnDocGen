# Модуль `release.py`


Этот модуль предоставляет функции для работы с zest.releaser при выпуске новых версий библиотеки qrcode.

**Содержание:**

- [Функции](#функции)

## Функции

### `def update_manpage(data)`

Обновляет версию в документе manpage.

**Параметры:**

- `data` (`dict`) — данные для обновления, содержит ключи "name" и "new_version"

**Побочные эффекты:**

Изменяет содержимое файла manpage на диске.

**Примеры:**

```python
update_manpage({"name": "qrcode", "new_version": "1.2.3"})
```
