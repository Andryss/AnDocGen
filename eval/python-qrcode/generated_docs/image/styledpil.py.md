# Модуль `image/styledpil.py`


Модуль для работы с изображениями с использованием PIL и стилизацией.

**Содержание:**

- [Классы](#классы)

## Классы

### `class StyledPilImage(qrcode.image.base.BaseImageWithDrawer)`

Класс для создания стилизованных изображений с использованием PIL, по умолчанию формат PNG.

**Поля:**

- `color_mask` (`QRColorMask`) — —

**Наследование:**

Класс наследует от `qrcode.image.base.BaseImageWithDrawer`.

#### Методы

##### `__getattr__(name)`

Возвращает атрибут из внутреннего объекта PIL Image.

**Параметры:**

- `name` (`str`) — имя атрибута, который нужно вернуть

##### `__init__(*args, **kwargs)`

Инициализирует объект `StyledPilImage`.

**Параметры:**

- `*args` — позиционные аргументы
- `**kwargs` — именованные аргументы

**Побочные эффекты:**

Инициализирует объект `StyledPilImage` с параметрами, переданными через `*args` и `**kwargs`.

**Примеры:**

```python
image = StyledPilImage(color_mask=SolidFillColorMask(), embedded_image_path="path/to/image.jpg")
```

##### `draw_embedded_image()`

Отрисовывает встроенные изображения на изображении.

**Побочные эффекты:**

Изменяет исходное изображение, добавляя встроенное изображение.

**Примеры:**

```python
styled_image = StyledPilImage()
styled_image.draw_embedded_image()
```

##### `draw_embeded_image()`

Не документировано.

**Возвращаемое значение:**

- `None` — возвращаемое значение не определено

##### `drawrect(row, col)`

Метод не используется.

**Параметры:**

- `row` — —
- `col` — —

##### `init_new_image()`

Инициализирует новый изображение с использованием цветового маски.

##### `new_image(**kwargs)`

Создает новый изображение с заданными параметрами.

**Параметры:**

- `**kwargs` — —

**Возвращаемое значение:**

- `Image` — новое изображение

**Примеры:**

```python
new_image = StyledPilImage.new_image()
```

##### `process()`

Обрабатывает изображение с учетом маски цветов и встраиваемого изображения.

**Побочные эффекты:**

Изменяет само изображение, применяя маску цветов и вставляя встраиваемое изображение.

**Примеры:**

```python
image = StyledPilImage()
image.process()
```

##### `save(stream, format = None, **kwargs)`

Сохраняет изображение в файл или поток.

**Параметры:**

- `stream` (`Any`) — файл или объект для записи изображения.
- `format` (`str, optional`) — формат изображения. Если не указан, будет использован формат, определенный атрибутом `kind`.
- `**kwargs` — дополнительные аргументы, передаваемые методу `save` библиотеки PIL.

**Побочные эффекты:**

Изменяет содержимое объекта `stream`, в который записывается изображение.

**Примеры:**

```python
image.save(open("output.png", "wb"))
```
