# Модуль `image/styles/moduledrawers/__init__.py`


Модуль для ленивой загрузки рисовальщиков PIL с предупреждением устаревания.

В настоящее время импорт рисовальщика PIL из этого модуля разрешен для обратной совместимости, но будет вызывать предупреждение DeprecationWarning. Это будет удалено в версии 9.0.

**Содержание:**

- [Функции](#функции)

## Функции

### `def __getattr__(name)`

Динамический импорт с предупреждением об устаревшем использовании для рисовалок модулей PIL.

**Параметры:**

- `name` (`str`) — имя атрибута для импорта

**Исключения:**

- `AttributeError` — если запрошенный атрибут не существует в модуле

**Побочные эффекты:**

Устанавливает предупреждениеDeprecationWarning, если атрибут является одной из устаревших рисовалок PIL.

**Примеры:**

```python
drawer = __getattr__("CircleModuleDrawer")
```
