# Модуль `image/styles/moduledrawers/base.py`


Модуль `image/styles/moduledrawers/base.py` содержит базовый класс для отрисовки модулей.

- [Классы](#классы)

## Классы

### `class QRModuleDrawer(abc.ABC)`

Класс для рисования модулей QR-кода на изображениях.

**Наследование:**

- `abc.ABC` — Базовый абстрактный класс Python, который требует реализации всех его абстрактных методов.

#### Методы

##### `__init__(**kwargs)`

Инициализирует экземпляр класса QRModuleDrawer.

**Параметры:**

- `**kwargs` — —

##### `drawrect(box, is_active) -> None`

Рисует прямоугольник на основе предоставленного бокса и статуса активности.

**Параметры:**

- `box` (`any`) — объект, содержащий информацию о размерах и позиции прямоугольника
- `is_active` (`bool`) — флаг, указывающий, является ли прямоугольник активным

##### `initialize(img: 'BaseImage') -> None`

Инициализирует объект QRModuleDrawer с изображением.

**Параметры:**

- `img` (`BaseImage`) — изображение для инициализации

**Примеры:**

```python
drawer = QRModuleDrawer()
drawer.initialize(my_image)
```
