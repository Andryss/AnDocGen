# Модуль `image/pil.py`


Модуль для работы с изображениями, используя библиотеку PIL.

**Содержание:**

- [Классы](#классы)

## Классы

### `class PilImage(qrcode.image.base.BaseImage)`

Класс для создания изображений QR-кодов с использованием библиотеки PIL.

**Наследование:**

qrcode.image.base.BaseImage

#### Методы

##### `__getattr__(name)`

Возвращает атрибут объекта `_img`.

**Параметры:**

- `name` (`str`) — имя атрибута, который нужно получить

**Возвращаемое значение:**

- `N` — /A

**Примеры:**

```python
image = PilImage()
attr_value = image.__getattr__('mode')
```

##### `drawrect(row, col)`

Отрисовывает прямоугольник на изображении.

**Параметры:**

- `row` (`int`) — строка начала прямоугольника
- `col` (`int`) — столбец начала прямоугольника

**Побочные эффекты:**

Изменяет изображение, на котором вызывается метод.

**Примеры:**

```python
# Пример использования
img = PilImage()
img.drawrect(10, 20)
```

##### `new_image(**kwargs)`

Создает новый изображение с заданными параметрами.

**Параметры:**

- `**kwargs` — —

**Возвращаемое значение:**

- `PIL.Image` — созданный объект изображения

**Исключения:**

- `ImportError` — если библиотека PIL не найдена

**Примеры:**

```python
img = PilImage.new_image(back_color="white", fill_color="black")
```

##### `save(stream, format = None, **kwargs)`

Сохраняет изображение в указанный поток.

**Параметры:**

- `stream` (`file`) — файловый объект для записи изображения
- `format` (`str, опционально`) — формат изображения, по умолчанию используется формат, указаный при создании изображения
- `**kwargs` — —

**Побочные эффекты:**

Изменяет поток, в который записывается изображение.

**Примеры:**

```python
from PIL import Image
img = Image.new("RGB", (100, 100))
with open("output.png", "wb") as f:
    img.save(f, format="PNG")
```
