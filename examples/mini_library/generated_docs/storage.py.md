# Модуль `storage.py`


Модуль `storage.py` содержит класс для хранения заказов в памяти процесса с использованием словаря.

**Содержание:**

- [Классы](#классы)

## Классы

### `class InMemoryStorage`

Класс для хранения заказов в памяти процесса с использованием словаря.

#### Методы

##### `__init__() -> None`

Инициализирует экземпляр класса `InMemoryStorage`, создавая пустой словарь для хранения заказов и начальное значение для следующего уникального идентификатора.

**Примеры:**

```python
storage = InMemoryStorage()
```

##### `create(customer: str) -> Order`

Создает новый заказ для указанного клиента и возвращает его.

**Параметры:**

- `customer` (`str`) — имя клиента

**Возвращаемое значение:**

- `Order` — созданный заказ

**Примеры:**

```python
storage = InMemoryStorage()
order = storage.create(customer="John Doe")
```

##### `get(order_id: int) -> Order`

Возвращает заказ по заданному идентификатору.

**Параметры:**

- `order_id` (`int`) — уникальный идентификатор заказа

**Возвращаемое значение:**

- `Order` — найденный заказ

**Исключения:**

- `NotFoundError` — если заказ с указанным идентификатором не существует

**Примеры:**

```python
order = InMemoryStorage.get(123)
```

##### `list_ids() -> list[int]`

Возвращает отсортированный список идентификаторов заказов.

**Возвращаемое значение:**

- `list[int]` — отсортированный список идентификаторов заказов

**Примеры:**

```python
storage = InMemoryStorage()
storage._orders[123] = Order(...)
storage._orders[456] = Order(...)
print(storage.list_ids())  # Output: [123, 456]
```
