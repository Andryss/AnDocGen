# Модуль `__init__.py`


Мини-библиотека с публичным API.

**Экспорт:**

- `Item` (`class`) — Класс для представления товара.
- `Order` (`class`) — Класс для представления заказа.
- `OrderService` (`class`) — Сервис для управления заказами.
- `InMemoryStorage` (`class`) — Стандартная реализация хранилища в памяти.
- `Settings` (`class`) — Настройки приложения.
- `MiniLibraryError` (`exception`) — Основное исключение библиотеки.
- `NotFoundError` (`exception`) — Исключение, возникающее при отсутствии ресурса.
- `ValidationError` (`exception`) — Исключение, возникающее при ошибке валидации данных.
- `main` (`function`) — Основная функция для запуска командной строки.
