**Содержание:**

- [Функции](#функции)

## Функции

### `def format_items(*labels: str) -> str`

Соединяет переменное количество меток элементов.

**Параметры:**

- `*labels` (`str`) — переменное количество меток элементов для соединения

**Возвращаемое значение:**

- `str` — соединенная строка меток элементов

**Примеры:**

```python
format_items('apple', 'banana', 'cherry')
# apple, banana, cherry
```

### `def format_price(amount: float, currency: str = 'USD') -> str`

Форматирует сумму денег с символом валюты.

**Параметры:**

- `amount` (`float`) — сумма денег
- `currency` (`str, optional`) — валюта (по умолчанию 'USD')

**Возвращаемое значение:**

- `str` — отформатированная строка с суммой и валютой

**Примеры:**

```python
format_price(123.456)  # 'USD 123.46'
format_price(789, 'EUR')  # 'EUR 789.00'
```
