# Модуль `plugins/base.py`


Базовые классы для плагинов в проекте mini_library.

**Содержание:**

- [Классы](#классы)

## Классы

### `class PluginBase(ABC)`

Базовый класс для экспортировальных плагинов. Наследуется от ABC и определяет методы `__init__` и `export`.

**Наследование:**

- `ABC`

#### Методы

##### `__init__(*args: Any, **kwargs: Any) -> None`

Инициализирует экземпляр базового плагина с опциями.

**Параметры:**

- `*args` (`Any`) — переменное количество позиционных аргументов
- `**kwargs` (`Any`) — переменное количество именованных аргументов

**Примеры:**

```python
plugin = PluginBase(options={'key': 'value'})
```

##### `export(data: dict[str, Any]) -> str`

Сериализует данные в строковое представление.

**Параметры:**

- `data` (`dict[str, Any]`) — данные для сериализации

**Возвращаемое значение:**

- `str` — строковое представление данных

**Примеры:**

```python
plugin = PluginBase()
result = plugin.export({"key": "value"})
print(result)  # Пример вывода: {"key": "value"}
```
