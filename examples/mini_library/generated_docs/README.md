# mini_library


## Корневые модули

| Модуль | Описание |
|--------|--------------|
| [__init__.py](__init__.py.md) | Мини-библиотека с публичным API. |
| [cli.py](cli.py.md) | Команда-строка для мини-библиотеки с базовыми операциями. |
| [config.py](config.py.md) | Модуль для управления настройками приложения `mini_library`. |
| [exceptions.py](exceptions.py.md) | Модуль исключений для мини-библиотеки `mini_library`. |
| [models.py](models.py.md) | Модуль `models.py` содержит определения данных для предметов и заказов в мини-библиотеке. |
| [services.py](services.py.md) | Модуль `services.py`, содержащий бизнес-логику для управления заказами. |
| [storage.py](storage.py.md) | Модуль `storage.py` содержит класс для хранения заказов в памяти процесса с использованием словаря. |
