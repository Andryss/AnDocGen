#### Методы

##### `total() -> float`

Вычисляет общую сумму заказа на основе цены и количества каждого товара в списке товаров.

**Возвращаемое значение:**

- `float` — сумма всех товаров в заказе

**Примеры:**

```python
order = Order(items=[Item(price=10.0, quantity=2), Item(price=5.0, quantity=3)])
print(order.total())  # Output: 35.0
```
