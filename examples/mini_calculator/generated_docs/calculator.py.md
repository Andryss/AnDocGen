# Модуль `calculator.py`


Простой модуль калькулятора с базовыми операциями сложения и деления.

**Содержание:**

- [Классы](#классы)
- [Функции](#функции)

## Классы

### `class Calculator`

Класс для базовых арифметических операций.

#### Методы

##### `__init__(precision: int = 2) -> None`

Инициализирует объект класса Calculator с заданной точностью.

**Параметры:**

- `precision` (`int`) — точность, по умолчанию 2

**Примеры:**

```python
calc = Calculator(precision=3)
```

##### `format_result(value: float) -> str`

Форматирует результат числа с заданной точностью.

**Параметры:**

- `value` (`float`) — число для форматирования

**Возвращаемое значение:**

- `str` — отформатированное число в виде строки с заданной точностью

**Примеры:**

```python
result = Calculator().format_result(123.456789)
print(result)  # Output: 123.457
```

##### `multiply(x: int, y: int) -> int`

Умножает два числа и возвращает результат.

**Параметры:**

- `x` (`int`) — первое число
- `y` (`int`) — второе число

**Возвращаемое значение:**

- `int` — произведение `x` и `y`

**Примеры:**

```python
result = Calculator.multiply(3, 4)  # Возвращает 12
```

## Функции

### `def add(a: float, b: float) -> float`

Складывает два числа и возвращает результат.

**Параметры:**

- `a` (`float`) — первое слагаемое
- `b` (`float`) — второе слагаемое

**Возвращаемое значение:**

- `float` — сумма двух чисел

**Примеры:**

```python
result = add(2.5, 3.7)
print(result)  # Output: 6.2
```

### `def divide(a: float, b: float) -> Optional[float]`

Делит два числа и возвращает результат. Если делитель равен нулю, возвращает `None`.

**Параметры:**

- `a` (`float`) — делимое
- `b` (`float`) — делитель

**Возвращаемое значение:**

- `Optional[float]` — частное или `None`, если делитель равен нулю

**Граничные случаи:**

- Если делитель равен нулю, функция возвращает `None`.

**Примеры:**

```python
result = divide(10, 2)  # result == 5.0
result = divide(10, 0)  # result == None
```
