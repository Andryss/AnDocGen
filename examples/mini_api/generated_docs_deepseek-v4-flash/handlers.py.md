### `class UserHandler`

Класс для обработки операций API, связанных с пользователями.

#### Методы

##### `__init__(storage: InMemoryStorage) -> None`

Конструктор класса `UserHandler`. Инициализирует экземпляр, сохраняя ссылку на переданное хранилище.

**Параметры:**

- `storage` (`InMemoryStorage`) — хранилище для работы с пользователями

**Побочные эффекты:**

Сохраняет переданное хранилище в приватном атрибуте `_storage`.

**Примеры:**

```python
from .storage import InMemoryStorage

storage = InMemoryStorage()
handler = UserHandler(storage)
```

##### `get_user(user_id: int) -> ApiResponse`

Метод получает пользователя из хранилища по его идентификатору. Если пользователь найден, возвращает ответ с кодом 200 и данными пользователя; в противном случае — ответ с кодом 404 и сообщением об ошибке.

**Параметры:**

- `user_id` (`int`) — идентификатор пользователя

**Возвращаемое значение:**

- `ApiResponse` — ответ с кодом 200 и телом, содержащим данные пользователя, если он найден; иначе ответ с кодом 404 и телом `{"error": "not found"}`

**Граничные случаи:**

Если пользователь с указанным `user_id` отсутствует в хранилище, метод возвращает `ApiResponse` с кодом 404.

**Примеры:**

```python
# Пользователь найден
response = handler.get_user(42)
# response.status == 200, response.body == данные пользователя

# Пользователь не найден
response = handler.get_user(999)
# response.status == 404, response.body == {"error": "not found"}
```

##### `create_user(name: str, email: str) -> ApiResponse`

Создаёт нового пользователя: нормализует email, сохраняет данные в хранилище и возвращает ответ с идентификатором созданной записи.

**Параметры:**

- `name` (`str`) — имя пользователя
- `email` (`str`) — адрес электронной почты (будет нормализован перед сохранением)

**Возвращаемое значение:**

- `ApiResponse` — объект ответа с HTTP-статусом 201 и телом `{"id": <идентификатор>}`

**Побочные эффекты:**

Создаёт запись в хранилище `self._storage` и генерирует новый идентификатор пользователя.

**Примеры:**

```python
handler = UserHandler(storage)
response = handler.create_user("Иван", "ivan@example.com")
# response.status == 201
# response.body == {"id": 42}
```

- `normalize_email` — нормализация адреса электронной почты
- `InMemoryStorage.create` — метод создания записи в хранилище
