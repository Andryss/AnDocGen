# mini_api

Документация проекта, сгенерированная AnDocGen.

## Корневые модули

| Модуль | Описание |
|--------|--------------|
| [__init__.py](__init__.py.md) | Модуль, содержащий мини‑пример API пакета. |
| [handlers.py](handlers.py.md) | Модуль содержит простые обработчики HTTP‑подобного API, включая классы ApiResponse и UserHandler, использующие InMemoryStorage для работы с пользователями. |
| [storage.py](storage.py.md) | Модуль хранит в памяти ключ-значение. |
