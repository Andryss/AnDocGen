## Классы

### `class InMemoryStorage`

Простой в памяти хранилище для демонстрационных целей.

#### Методы

##### `__init__() -> None`

Инициализирует объект InMemoryStorage, создавая внутренний словарь для хранения данных и устанавливая счётчик следующего идентификатора.

**Побочные эффекты:**

Создаёт атрибуты _data и _next_id, которые используются для хранения данных и генерации уникальных идентификаторов.

**Примеры:**

storage = InMemoryStorage()

##### `create(payload: dict[str, Any]) -> int`

Создаёт новый элемент в памяти и возвращает его уникальный идентификатор.

**Параметры:**

- `payload` (`dict[str, Any]`) — пакет данных, который будет сохранён

**Возвращаемое значение:**

- `int` — уникальный идентификатор созданного элемента

**Побочные эффекты:**

Изменяет внутреннее состояние: увеличивает счётчик `_next_id` и сохраняет `payload` в словаре `_data` под новым ключом.

**Примеры:**

storage = InMemoryStorage()
item_id = storage.create({"name": "example"})
print(item_id)  # 0

##### `get(item_id: int) -> dict[str, Any] | None`

Метод возвращает элемент из внутреннего словаря по заданному идентификатору.

**Параметры:**

- `item_id` (`int`) — идентификатор запрашиваемого элемента

**Возвращаемое значение:**

- `dict[str, Any] | None` — словарь элемента, если он найден, иначе `None`

**Граничные случаи:**

Если `item_id` отсутствует в хранилище, возвращается `None`.

**Примеры:**

```python
storage = InMemoryStorage()
storage._data = {1: {'name': 'item1'}, 2: {'name': 'item2'}}
print(storage.get(1))  # {'name': 'item1'}
print(storage.get(3))  # None
```

## Функции

### `def normalize_email(email: str) -> str`

Функция нормализует адрес электронной почты для хранения: удаляет пробелы по краям и приводит его к нижнему регистру.

**Параметры:**

- `email` (`str`) — адрес электронной почты, который нужно нормализовать

**Возвращаемое значение:**

- `str` — нормализованный адрес электронной почты

**Примеры:**

normalize_email("  EXAMPLE@Domain.COM  ")  # "example@domain.com"
normalize_email("User@Example.org")  # "user@example.org"
