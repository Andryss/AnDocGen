# Модуль `storage.py`


Модуль хранения данных в оперативной памяти с возможностью нормализации адресов электронной почты.

**Содержание:**

- [Классы](#классы)
- [Функции](#функции)

## Классы

### `class InMemoryStorage`

Класс для простого хранения данных в оперативной памяти для демонстрационных целей.

#### Методы

##### `__init__() -> None`

Инициализирует экземпляр класса InMemoryStorage.

**Возвращаемое значение:**

- `None` — инициализация происходит на месте без возврата значения

##### `create(payload: dict[str, Any]) -> int`

Создаёт новый элемент в хранилище и возвращает его идентификатор.

**Параметры:**

- `payload` (`dict[str, Any]`) — данные для нового элемента

**Возвращаемое значение:**

- `int` — идентификатор созданного элемента

##### `get(item_id: int) -> dict[str, Any] | None`

Возвращает данные по указанному идентификатору из хранилища.

**Параметры:**

- `item_id` (`int`) — идентификатор элемента

**Возвращаемое значение:**

- `dict[str, Any] | None` — данные элемента или `None`, если элемент не найден

## Функции

### `def normalize_email(email: str) -> str`

Приводит адрес электронной почты к нижнему регистру и удаляет начальные и конечные пробелы.

**Параметры:**

- `email` (`str`) — адрес электронной почты для нормализации

**Возвращаемое значение:**

- `str` — нормализованный адрес электронной почты

**Примеры:**

```python
normalize_email("USER@EXAMPLE.COM")
```
